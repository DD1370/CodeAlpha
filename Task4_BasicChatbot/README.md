# 🤖 Basic Chatbot

A simple rule-based chatbot built with Python. The chatbot responds to predefined user inputs and demonstrates the use of functions, loops, conditional statements, and user interaction.

## 📌 Features

* Responds to basic greetings and questions
* Uses predefined replies
* Continuous conversation using a loop
* Exits when the user types `bye`
* Beginner-friendly Python project

## 🛠️ Technologies Used

* Python 3

## 🧠 Concepts Used

* Functions
* Loops (`while`)
* Conditional Statements (`if-elif-else`)
* User Input (`input()`)
* Output (`print()`)

## 📖 Learning Outcomes

This project helps in understanding:

* Basic chatbot logic
* User interaction in Python
* Decision-making with conditional statements
* Using loops to maintain conversations
* Organizing code with functions
