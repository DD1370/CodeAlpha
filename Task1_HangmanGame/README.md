# 🎮 Hangman Game

A simple console-based Hangman game built with Python. The player must guess the hidden word one letter at a time before running out of lives.

## 📌 Features

* Randomly selects a word from a predefined list
* Letter-by-letter guessing
* Input validation
* Tracks previously guessed letters
* 6 lives per game
* Win and lose conditions
* Play Again option
* Beginner-friendly Python project

## 🛠️ Technologies Used

* Python 3
* Random Module


## 🎯 How to Play

1. The game randomly chooses a hidden word.
2. Guess one letter at a time.
3. Correct guesses reveal letters in the word.
4. Wrong guesses reduce your remaining lives.
5. You have 6 lives to guess the word.
6. Win by revealing all letters before running out of lives.

## 🧠 Concepts Used

* Variables
* Lists
* Strings
* Loops (`while`)
* Conditional Statements (`if-else`)
* Input Validation
* Random Selection
* Game Logic
