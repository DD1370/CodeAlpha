# 📈 Stock Portfolio Tracker

A simple Python-based Stock Portfolio Tracker that allows users to enter stock symbols and quantities, calculates the total investment value, and saves the portfolio summary to a text file.

## 📌 Features

* Track multiple stocks and quantities
* Uses a predefined stock price dictionary
* Calculates individual stock values
* Calculates total portfolio value
* Handles invalid stock symbols
* Saves portfolio summary to a `.txt` file
* Beginner-friendly console application

## 🛠️ Technologies Used

* Python 3
* Dictionaries
* Loops
* Conditional Statements
* File Handling

## 🎯 How to Use

1. Enter a stock symbol from the available list.
2. Enter the quantity of shares.
3. Repeat until finished.
4. Type `done` to view your portfolio summary.
5. The program calculates the total investment value.
6. A summary is automatically saved to `portfolio_summary.txt`.
